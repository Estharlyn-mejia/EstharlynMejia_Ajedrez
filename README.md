# EstharlynMejia_Ajedrez

Proyecto de ajedrez desarrollado en JavaFX.

## Descripción
Este proyecto implementa un juego de ajedrez con interfaz gráfica utilizando JavaFX.

## Estructura
- src: código fuente
- resources: imágenes del tablero y piezas

## Estructura
